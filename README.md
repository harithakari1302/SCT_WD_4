# To-Do Web App

A responsive To-Do List application developed using HTML, CSS, and JavaScript.

## Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Set date and time for tasks
- Save tasks using Local Storage
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript
